## 2026-06-15 题12 ReAct 框架的核心特点

来源：用户提供 30 题随机组卷

题目：
> 在Agent架构中，ReAct(Reasoning + Acting)框架的核心特点是什么？

选项：
- A. 只进行推理不执行动作
- B. 将推理(Thought)和工具调用(Action)交替进行，推理结果指导下一步动作
- C. 同时执行所有可能的动作，选择最优结果
- D. 使用强化学习训练Agent的动作策略

结论：
正确选项：B

大白话解析：
ReAct 就是让 Agent 一边想、一边行动。它先根据问题进行推理，决定下一步要调用什么工具或执行什么动作，再根据观察结果继续推理和行动。

易错点：
- A 只推理不行动，不是 ReAct。
- C 同时执行所有动作成本高，也不是 ReAct 定义。
- D ReAct 是一种推理-行动框架，不等同于强化学习训练方法。

知识点总结：
- ReAct = Reasoning + Acting，推理和工具调用交替进行。
